## What is Flow Browser?

Flow Browser is a lightweight, feature-rich web browser that combines the power of Chromium with a clean, modern interface. Built on Electron, it offers full support for Chrome extensions, making it a versatile alternative to mainstream browsers.

<Note>
  Flow Browser is currently in beta (v0.10.1) and actively under development. Join us in shaping the future of privacy-focused browsing!
</Note>

## Privacy & Security

Flow Browser prioritizes your privacy:

- **External Application Prompts**: Always asks before opening external applications
- **Native Ad Blocking**: Blocks ads and trackers at the network level using [@ghostery/adblocker-electron](https://www.npmjs.com/package/@ghostery/adblocker-electron)
- **No Telemetry by Default**: Your browsing data stays on your device
- **Open Source**: Full transparency with GPL-3.0 license

## What Makes Flow Different?

<Steps>
  <Step title="Minimalistic Design">
    Inspired by Arc Browser and Zen Browser, Flow features a clean, distraction-free interface with a customizable sidebar and optional topbar.
  </Step>
  
  <Step title="Workspace Organization">
    Unique combination of Profiles (for different identities) and Spaces (for task organization) helps you keep work, personal, and hobby browsing completely separate.
  </Step>
  
  <Step title="Developer-Friendly">
    Built on open web standards with full Chrome DevTools support, extension APIs, and an architecture designed for customization.
  </Step>
  
  <Step title="Resource Efficient">
    Sleep tabs feature and optimized rendering help reduce memory usage compared to traditional browsers.
  </Step>
</Steps>
